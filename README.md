# gempro-notebook
Materiais impressos para a maratona de programação.

## Builds

- `./build.sh all`: gera `out/gempro.pdf` e `out/gempro-extended.pdf`
- `./build.sh core`: gera apenas `out/gempro.pdf`, com explicações enxutas
- `./build.sh extended`: gera apenas a versão estendida, com explicações completas
- `./build.sh legacy`: gera `out/gempro-notebook.pdf` apontando para a versão estendida
