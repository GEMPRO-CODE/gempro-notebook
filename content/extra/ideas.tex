\subsection{Gerais}
\begin{itemize}
	\item Em problemas envolvendo somas em intervalos de arrays, pode ser mais fácil trabalhar com o array de somas de prefixos.
	\item Muitas DPs de bitmask podem ser simplificadas para usar apenas alguma propriedade da bitmask. (Número de bits, etc)
	\item Se o problema descreve um processo que parece complicado, talvez seja mais fácil ver o processo ao contrário.
\end{itemize}
\subsection{Calculando somas e integrais}
\begin{itemize}
	\item Verificar se existe uma fórmula fechada
	\item Inverter a ordem da soma (em somatórios duplos)
	\item Transformar em somatório duplo e inverter a ordem
	\item Verificar se a soma pode ser calculada de um jeito eficiente (sem ser uma fórmula fechada)
	\item No caso de calcular várias somas, verificar se existe uma relação entre elas (Calcular uma pode ajudar a calcular outra)
\end{itemize}
\subsection{Problemas de Otimização}
\begin{itemize}
	\item Encontrar um upper bound e provar que ele sempre é alcançável.
	\item Encontrar uma construção boa, e provar que não é possível fazer melhor.
	\item Escrever um brute-force e testar com casos pequenos para encontrar algum padrão.
\end{itemize}
