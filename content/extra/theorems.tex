\textbf{Teorema de Erdos-Szekeres: } Toda sequência de $(r - 1)(s - 1) + 1$ inteiros distintos possui uma subsequência crescente de tamanho $r$ ou uma subsequência decrescente de tamanho $s$. \\
\textbf{O valor esperado é linear: }
\[\mathbb{E}(aX + bY) = a\mathbb{E}(X) + b\mathbb{E}(Y)\]
\textbf{Teorema EGZ: } Dados $2n - 1$ inteiros $a_1, a_2, \dots, a_{2n-1}$, existem $n$ índices $i_1, i_2, \dots, i_n$ tais que \[a_{i_1} + a_{i_2} + \cdots + a_{i_n} = 0 \pmod{n}\]
