\templateentry
{Suffix Array}
{\texttt{sufArr(s)} retorna os índices dos sufixos em ordem lexicográfica.}
{Suffix array por doubling. Interface: chame \texttt{sufArr(s)}; o retorno é um vetor \texttt{sa} em que \texttt{sa[i]} é o índice de início do $i$-ésimo sufixo na ordem lexicográfica. Complexidade: $O(n \log n)$.}
{strings/suffix-array.cpp}

\templateentry
{Maior prefixo comum (LCP)}
{\texttt{lcpArr(s, sa)} calcula o LCP entre sufixos adjacentes.}
{Algoritmo de Kasai. Interface: passe a string \texttt{s} e o suffix array \texttt{sa}; o retorno é um vetor \texttt{lcp} onde \texttt{lcp[i]} é o maior prefixo comum entre \texttt{sa[i]} e \texttt{sa[i + 1]}. Complexidade: $O(n)$.}
{strings/lcp.cpp}

\templateentry
{KMP}
{\texttt{kmp(s)} retorna a prefix function em $O(n)$.}
{KMP/prefix function. Interface: chame \texttt{kmp(s)}; o retorno é o vetor \texttt{pre}, onde \texttt{pre[i]} é o tamanho do maior prefixo próprio de \texttt{s} que também é sufixo de \texttt{s[0..i]}. Útil para matching, periodicidade e automatos simples.}
{strings/kmp.cpp}
