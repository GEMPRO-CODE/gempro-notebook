\subsection{Suffix Array}
Dada uma string $S$, ordena todos os sufixos em ordem lexicográfica. Retorna um array $SA$ tal que \[SA_i = \text{índice do começo do } i\text{-ésimo sufixo em ordem lexicográfica}\]
Complexidade: $O(n\log(n))$, onde $n$ é o tamanho da string.

\subsection{KMP}
Dada uma string $S$, calcula um array $pre$ de forma que \[pre_i = \text{ maior valor de } k < i \text{ tal que } S_0 S_1 \cdots S_{k - 1} = S_{i - k + 1}S_{i - k + 2} \cdots S_i \]
Complexidade: $O(n)$.
